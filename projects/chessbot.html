Chessbot

01/2025 - Present

Overview

An automated chess-playing robot using an electromagnet gantry system and MQTT host communication architecture. Developed LCD UI with C using the Little Video Graphics Library (LVGL) video graphics framework to enable direct user control for the robot states by pausing and homing the gantry. More documentation on Chessbot's development page on UT IEEE RAS's website .
